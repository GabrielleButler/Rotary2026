# Rotary2026

Bluetooth Low Energy (BLE) communication software developed for the Rotary 2026 drone competition.

## Features

- BLE communication
- Drone command transmission
- Flight control utilities
- Competition-ready code

## Requirements

- VS Code
- Arduino IDE / PlatformIO (whichever you're using)
- Required BLE libraries

## Installation

Clone the repository:

```bash
git clone https://github.com/GabrielleButler/Rotary2026.git
```

Open the project and upload the firmware to the drone hardware.
